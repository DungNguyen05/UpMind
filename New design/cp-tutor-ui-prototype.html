CP-Tutor — 10 màn hình sản phẩm

Mỗi trang là một file HTML riêng, có thể mở/view độc lập. Thiết kế desktop-first, hỗ trợ tablet, giao diện tiếng Việt và tối ưu cho học C++ competitive programming.

10 trang C++17 AI panel
01 /login

Đăng nhập

Form auth, show/hide password, loading và lỗi.

 02 /register

Đăng ký

Validation username, strength password, confirm password.

 03 /problems

Danh sách bài tập

Search, filter độ khó, multi-tag filter, empty state.

 04 /problems/[slug]

Trang giải bài

3 cột: đề bài, editor C++ và AI Mentor với submit simulation.

 05 /submissions

Lịch sử nộp bài

Bảng submissions có lọc theo bài và verdict.

 06 /submissions/[id]

Chi tiết submission

Code read-only, runtime/memory và AI feedback.

 07 /admin/problems

Admin — Bài tập

Quản lý publish/draft, edit/delete và stats.

 08 /admin/problems/new

Admin — Tạo/Sửa bài

Form dài, test cases, live Markdown preview.

 09 /admin/users

Admin — Người dùng

Search user, role badge, đổi role inline.

 10 404
